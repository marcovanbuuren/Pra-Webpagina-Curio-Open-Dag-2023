Dit zijn de vakken die je gaat krijgen in het eerste schooljaar.

Nederlands

Nederlands is een taal die je nog steeds krijgt. Het is een belangrijke taal als je gaat zoeken naar een stage, maar ook voor de opleiding bv. bij een cv schrijven in het 2de leerjaar

Engels

Engels is een taal die geld als wereldtaal. Het is belangrijk taal in software development. De taal wordt gezien als communicatietaal tussen sorftware developers. Ook zijn de handleidingen die je kan vinden over een bepaalde programmeertaal in het engels.

Rekenen

Rekenen is een vak, waarvan je leert hoe je in het dagelijks leven moet uitrekenen met bepaalde situaties die je later mogelijk gaat tegenkomen.

Digitale vaardigheden

Digitale vaardigheden is een vak, waarbij je leert kennis op te bouwen over bepaalde digitale vaardigheden en leert werken met bv. Word, Excel en Wordpress.

Pra

Praktijk is een vak, waarvoor je het geleerde werk die je tijdens de vakken hebt geleert samenvoegd tot een werken project.

Pro

Praktijkondersteuning is een vak, waarvan je de verschillende manieren van werken in een team tijdens een project leert die gelden bij software development.

Web

Website-development is een vak, waarvan je leert een website te maken en de dingen rondom. bv. de style, bestanden verwerken en een database aanmaken en beheren.

Nativ

Native-development, vroeger Win-development is een vak, waarvan je leert programmeren voor een windows-applicaties. Je leert onder anderen de programmeertaal python en c#.

Burgerschap

Burgerschap is een vak, waarvan je leert je rechten en plichten in de samenleving te begrijpen.

Mentooruur

Mentooruur is een uur, waarin je met jouw mentor de belangrijkste dingen over de opleiding doorneemt.

Algoritmiek

Algoritmiek is een vak, waarvan je leert hoe het algoritme werkt bij een computer.
